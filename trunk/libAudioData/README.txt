libAudioData v1.0.0 by David Starkweather
08/11/2014

------------------------------------------------------------

Simple interface to read audio data from files to a chosen
sample rate.  All stereo channels are merged into one channel.
All meta data is extracted as well. 

List of file formats supported: .wav, .mp3, .mp2, .amr, .aifc, .ogg, .flac
